Plexus is an Inversion of Control (IoC) container employed by Maven. Plexus components can be injected into Maven plugins. In addition, Plexus provides a number of components and utilities that can be of use when coding Maven plugins. Finally, Plexus provides a Maven plugin that can be used to automatically generate the metadata required by the Plexus container.
