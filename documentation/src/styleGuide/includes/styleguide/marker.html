The @CodeStyle annotation is used to define a code style based on code in the containing class. The specific code to be used as an example for that style must be bracketed by a pair of comment markers. Unless otherwise configured, the markers must comply with the default format: "--- StyleGuide <categoryId>.<topicId> ---", where <categoryId> and <topicId> are the values used for the categoryId and topicId annotation elements, respectively.
